# js-boilerplate

![npm](https://img.shields.io/npm/v/npm.svg) ![node (scoped with tag)](https://img.shields.io/node/v/@stdlib/stdlib/latest.svg)


Minimal boilerplate for rapid JavsScript ideation



---

### Available Scripts

```bash
Install modules:
$ yarn

```

`yarn start`

Runs a webpack dev server in development mode. Page reloads on edit

Open [http://localhost:8080](http://localhost:8080)

```bash

$ yarn start

```


`yarn build`

Builds production bundles in the `dist` folder

```bash

$ yarn build

```
